🚗 EV Data Visualization Dashboard (Tableau)
This project presents an interactive Tableau dashboard analyzing Electric Vehicle (EV) data.

Project Purpose
This project helps understand EV market trends and supports data-driven decision making.

Tools Used: 
1. Tableau
2. Excel / CSV dataset

Key Insights
•	EV adoption trends are increasing over time
•	Battery Electric Vehicles (BEV) vs Plug-in Hybrid Electric Vehicles (PHEV) comparison
•	Regional distribution of EV usage
•	Key patterns in EV growth and demand

📁 Project Files
•	EV Data Analysis.twbx → Tableau dashboard file
•	BEV Analysis Dashboard-Viz.png → BEV dashboard view
•	PHEV Analysis Dashboard-Viz.png → PHEV dashboard view
•	Overall EV Analysis Dashboard-Viz.png → Overall analysis

How to Use
•	Download the .twbx file
•	Open in Tableau Desktop-Explore interactive dashboards
